# Saminest v2

Vite + React + TypeScript rewrite of Saminest. See `docs/` for the product, architecture, database, and AI development documentation — read `docs/02_SystemDesign/Architecture.md` and `docs/04_Development/AI-Development.md` before making changes.

## Scripts

```bash
npm run dev         # start the dev server
npm run typecheck   # tsc project references, no emit
npm run test        # vitest run
npm run build       # typecheck + production build
npm run preview     # preview the production build
```
